Instructions

Step-by-step instructions to create a script on this site are listed below. If there are issues or bugs, contact person.

If you are creating a new year which does not have any scripts on the MOB site yet, click the "Add a new year button" and enter the year. This will generate the code for the link to the year at the top of the Scripts page as well as the container for the links to the scripts in that year. Copy the code snippets to their respective locations as directed.

  1. Create a show. Enter the script ID in the format shown (school abbreviation then year). This ID will connect the show link to the show itself, and must be entered in this form before completing "7. Create link to show." Enter the full name of the opposing school (don't need to go overboard though). Mark if the game was home or away. Enter the stadium or venue name, then the city and state. Enter the date of the show. Mark whether the result of the game was a win, loss, or tie, and the final score.

    If any of these fields are not applicable, for instance if we perform a show at a non-sporting event, leave the fields blank, and they will be left out of the code.

  2. Begin a script. Add the title of the show, if it has one. If the show has no title, leave the "Show Title" field blank, but still submit the form with the "Begin Script" button. This will add the title if there is one and create the table the script goes in. If the game you are recording has both a pregame and halftime that you wish to record, you can use this button again after "5. Finish script."

  3. Add a line. Specify the actor of the line—the announcer is saying something, the MOB is making a formation, etc. The "MOB" option is generally used to specify something the entire MOB says or does, as opposed to individual actions by SAs or formations. Leave blank if you want to add something only to the right side of the table. Under "What," enter whatever part of the script you are adding. Repeat as many times as necessary.

    (3.5. Add image). This will open a separate box where you can easily add an image to the script, such as a formation or whatever is on the videoboard.

  4. Add new show segment. Use this button to add a horizontal line beneath the last line you added. This generally corresponds to the end of playing a song. Repeat as many times as necessary.

    (4.5. Add epilogue). Sometimes a show might be noteworthy enough to merit an epilogue, which in the past has been used to discuss generally how the show went, any quirks, etc. Definitely optional. Do not add a new show segment before creating an epilogue. See Tulane 1989 for an example.

  5. Finish script. This button closes the table which contains the script. If another script follows the first, begin it after this button.

  6. Finalize script. This button closes off the entire section. Use after everything for the event has been added already.

  7. Create link to show. Enter the title to display in the link to the show. Enter the name of the school which is most commonly identified (see table below for common schools). Enter the abbreviated date as shown in the field. This link goes inside the code created by the "Add a new year button."

  8. Copy all code created to their appropriate places in the MOB website code and publish. You're done!

HTML Tips

This tool is programmed in pretty basic JavaScript and doesn't have many advanced functionalities. Some formatting keyboard shortcuts are available, but it's probably useful to learn a few common HTML tools to help with formatting.

The biggest HTML concept you should know is called a "tag." These are the things which appear in pairs surrounding bits of text or code, such as <div></div>. The tags operate on everything which is typed between them according to what kind of tags they are. For instance, using the underlining tag like this <u>Hello</u> will result in the text Hello.

Here are some common tags for formatting:

  • <u></u>: underline. You can open or close this tag with Ctrl + u while editing certain boxes.
  • <i></i>: italics. You can open or close this tag with Ctrl + i.
  • <b></b>: bold. You can open or close this tag with Ctrl + b.
  • <ul></ul>: Bullet points (like this list). Each bullet item should have the tags <li></li> surrounding it
  • <ol></ol>: Numbered list (like the instructions list above). Also needs <li></li> tags for each item
  • <p></p>:

    Denotes a paragraph which is placed on its own line.

    Consecutive paragraphs marked by this tag will have a small space between them

  • <br>: Line break. This tag does not come in a pair, and you just place it wherever you want the break to be.
    No extra space
    between lines here.
  • <img src="[image url]"/>: Image. Also does not come in a pair. See this link for image formatting details
  • <a href="[url]"></a>: Link to the url specified. Text between these tags is what the link shows up as, like above

You may also need some character codes for characters not on a standard keyboard. These are created with a & and a ; with the code for the character in the middle. Here are some common character codes:

  • &mdash; an em-dash (—)

These characters need to be replaced with their codes so HTML interprets them as text and not as a command:

  • &amp; &
  • &lt; <
  • &gt; >
